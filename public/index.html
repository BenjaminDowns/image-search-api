Bing Image Searcher

because who uses google anyway?

How to:

1) Go to https://bing-image-searcher.herokuapp.com/api/<your-search-words-here>

2) The image search returns a JSON object containing 10 results with the image's alt text (result.text), url (result.url), originating page (result.context), and url for the thumbnail (result.thumb).

3) options:

Offset: add '?offset=<number-from-1-to-10-here>'
each 'offset' is equal to one page later in the search results

Recent: get recent searches by going to https://bing-image-searcher.herokuapp.com/recent

Examples:

Request: https://bing-image-searcher.herokuapp.com/api/kittens?offset=2

Response: click here to see response

Request: https://bing-image-searcher.herokuapp.com/recent

Response: click here to see response
